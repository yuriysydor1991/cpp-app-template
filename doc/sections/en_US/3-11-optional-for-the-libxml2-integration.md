## Optional for the libxml2 integration

In order to enable the libxml2 integration in the template project install the system development package for the [libxml2](https://gitlab.gnome.org/GNOME/libxml2) library (GNU/Linux based):

```
sudo apt install -y libxml2-dev
```

For more details on how to enable the libxml2 target examine the [Enabling the libxml2](/doc/sections/en_US/5-20-enabling-libxml2.md) section.
