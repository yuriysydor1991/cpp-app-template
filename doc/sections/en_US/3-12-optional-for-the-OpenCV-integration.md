## Optional for the OpenCV integration

In order to enable the OpenCV integration in the template project install the system development packages for the [OpenCV](https://opencv.org/) library (GNU/Linux based):

```
sudo apt install -y libopencv-dev opencv-data
```

The `opencv-data` package ships the preinstalled Haar cascades which are commonly referenced from OpenCV-based applications.

For more details on how to enable the OpenCV target examine the [Enabling the OpenCV](/doc/sections/en_US/5-21-enabling-OpenCV.md) section.
