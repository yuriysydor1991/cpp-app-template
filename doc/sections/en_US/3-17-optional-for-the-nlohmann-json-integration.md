## Optional for the nlohmann JSON integration

In order to enable the nlohmann JSON integration in the template project install the system development package for the header-only [nlohmann JSON](https://github.com/nlohmann/json) library (GNU/Linux based):

```
sudo apt install -y nlohmann-json3-dev
```

For more details on how to enable the nlohmann JSON target examine the [Enabling the nlohmann JSON library](/doc/sections/en_US/5-18-enabling-the-nlohmann-json-library.md) section.
