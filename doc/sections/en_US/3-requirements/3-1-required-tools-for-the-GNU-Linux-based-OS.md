## Required tools for the GNU/Linux based OS

In order to build minimum template project install the GCC C++ compiler with CMake and Git.

```
sudo apt install -y git g++ cmake
```

The [SDL3](https://wiki.libsdl.org/SDL3/FrontPage) development packages are documented separately - see [Required packages for the SDL3 development](/doc/sections/en_US/3-requirements/3-12-required-for-the-SDL3-development.md).
