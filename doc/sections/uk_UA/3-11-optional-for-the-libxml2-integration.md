## Необов'язкові пакети для інтеграції libxml2

Для того щоб увімкнути інтеграцію libxml2 у шаблонному проекті необхідно встановити системний пакет розробника для бібліотеки [libxml2](https://gitlab.gnome.org/GNOME/libxml2) (для GNU/Linux ОС і подібних):

```
sudo apt install -y libxml2-dev
```

Більш детально про вмикання інтеграції libxml2 у секції [Вмикання інтеграції libxml2](/doc/sections/uk_UA/5-20-enabling-libxml2.md).
