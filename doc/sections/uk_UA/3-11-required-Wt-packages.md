## Обов'язкові пакунки для Wt

Для того щоб зробити [Wt C++ фреймворк](https://www.webtoolkit.eu/wt) доступним для розробки програм рекомендується звернутися до офіційної його сторінки розміщеної за адресою [https://www.webtoolkit.eu/wt/download](https://www.webtoolkit.eu/wt/download).

Необхідно звернутися до документації [Wt C++ фреймворку](https://www.webtoolkit.eu/wt) задля повного списку обов'язкових пакетів і інструментів.
