## Необов'язкові пакети для інтеграції OpenCV

Для того щоб увімкнути інтеграцію OpenCV у шаблонному проекті необхідно встановити системні пакети розробника для бібліотеки компʼютерного зору [OpenCV](https://opencv.org/) (для GNU/Linux ОС і подібних):

```
sudo apt install -y libopencv-dev opencv-data
```

Пакет `opencv-data` містить передвстановлені Haar-каскади, які часто використовуються застосунками на основі OpenCV.

Більш детально про вмикання інтеграції OpenCV у секції [Вмикання інтеграції OpenCV](/doc/sections/uk_UA/5-21-enabling-OpenCV.md).
