## Необов'язкові пакети для пакувальника snap

Для того щоб генерувати snap-пакет проекту необхідно встановити команду `snapcraft` (для GNU/Linux ОС і подібних):

```
sudo snap install --classic snapcraft
```

Старші випуски Ubuntu також містять `snapcraft` як deb-пакет, який можна встановити командою `sudo apt install -y snapcraft`. Звернись до [документації snapcraft](https://snapcraft.io/docs/installing-snapcraft) щодо рекомендованого способу встановлення для свого дистрибутиву.

Деталі у секції [Вмикання підтримки генерації snap-пакету](/doc/sections/uk_UA/5-22-enabling-snap-package-generation.md).
