## Необов'язкові пакети для інтеграції libcurl

Для того щоб увімкнути інтеграцію libcurl у шаблонному проекті необхідно встановити системний пакет розробника для бібліотеки [libcurl](https://curl.se/) (для GNU/Linux ОС і подібних):

```
sudo apt install -y libcurl4-openssl-dev
```

Альтернативні SSL-бекенди постачаються у вигляді `libcurl4-gnutls-dev` або `libcurl4-nss-dev` - обирай той, який відповідає решті системного стеку.

Більш детально про вмикання інтеграції libcurl у секції [Вмикання інтеграції libcurl](/doc/sections/uk_UA/5-14-enabling-libcurl.md).
