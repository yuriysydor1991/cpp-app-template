## Необов'язкові пакети для інтеграції nlohmann JSON

Для того щоб увімкнути інтеграцію nlohmann JSON у шаблонному проекті необхідно встановити системний пакет розробника для header-only бібліотеки [nlohmann JSON](https://github.com/nlohmann/json) (для GNU/Linux ОС і подібних):

```
sudo apt install -y nlohmann-json3-dev
```

Більш детально про вмикання інтеграції nlohmann JSON у секції [Вмикання інтеграції бібліотеки nlohmann JSON](/doc/sections/uk_UA/5-18-enabling-the-nlohmann-json-library.md).
