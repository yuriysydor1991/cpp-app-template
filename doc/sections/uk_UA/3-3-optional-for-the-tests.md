## Необов'язкові пакети для тестів

Якщо необхідно побудувати і виконати доступні тести необхідно встановити GTest. Якщо встановлена у системі версія GTest не буде знайдена, скрипти Meson-побудови проекту-шаблону спробують підтягнути її за наявного доступу до мережі Інтернет. Для встановлення системного пакету необхідно виконати команду:

```
sudo apt install -y googletest
```

Більш детально у секції [тести проекту](/doc/sections/uk_UA/4-4-project-tests.md).
