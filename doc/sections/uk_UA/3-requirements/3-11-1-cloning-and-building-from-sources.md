### Клонування і будування з вихідного джерельного коду

Для того щоб побудувати [Wt C++ фреймворк](https://www.webtoolkit.eu/wt) з вихідного джерельного коду розміщеного на репозиторію GitHub за адресою [https://github.com/emweb/wt](https://github.com/emweb/wt) необхідно його склонувати у локальну директорію цільової машини:

```
git clone https://github.com/emweb/wt.git
```

Далі слід дотриматись інструкції з README.md файлу [Wt C++ фреймворку](https://www.webtoolkit.eu/wt).
