### Встановлення за допомогою пакетного менеджера

Офіційна сторінка [Wt C++ фреймворку](https://www.webtoolkit.eu/wt) описує як встановити його використовуючи команду:

```
# може не спрацювати

sudo apt-get install witty witty-dev witty-doc witty-dbg witty-examples
```

**Не усі дистрибутиви ОС підтримують пакети Wt фреймворку**
