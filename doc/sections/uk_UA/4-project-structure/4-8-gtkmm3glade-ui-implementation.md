## Реалізування UI

Шаблонний файл проекту що знаходиться за шляхом `src/gtkmm3/ui/GtkmmWindow.ui` являється загальним дизайном візуальних інтерфейсів користувача для програми і містить реалізацію за умовчанням. Щоб реалізувати власний інтерфейс користувача необхідно відкрити файл `src/gtkmm3/ui/GtkmmWindow.ui` за допомогою програми [Glade](https://en.wikipedia.org/wiki/Glade_Interface_Designer) і замінити шаблон новими макетами.
