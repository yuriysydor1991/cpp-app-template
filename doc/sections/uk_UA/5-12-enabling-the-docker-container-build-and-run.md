## Вмикання підтримки запуску Docker-контейнера

Щоб увімкнути і запустити Docker контейнер разом з цільовим бінарним виконуваним файлом поточного проекту-шаблону необхідно встановити у значення `ON` змінну `ENABLE_DOCKER` за допомогою команди:

```
# в середині кореневої директорії проекту

meson setup build -DENABLE_DOCKER=true
```

Після чого можна запустити побудову контейнера Docker і наступний запуск цільового виконуваного файлу проекту за допомогою комнади (CMake-ціль `docker-single-run`):

```
# всередині кореневої директорії проекту

meson compile -C build docker-single-run 
```

Для деталей цілі побудови `docker-single-run` слід звернутись до файлу `meson/template-project-dockerers/meson.build`.
