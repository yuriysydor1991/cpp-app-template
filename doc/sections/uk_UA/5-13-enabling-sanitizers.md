## Вмикання перевірок під час виконання програми (sanitizers)

Для того щоб увімкнути додаткові перевірки під час виконання програми необхідно сконфігурувати проект з увімкненою опцією `ENABLE_SANITIZERS`:

```
# всередині кореневої директорії проекту

meson setup build -DENABLE_SANITIZERS=true
```

Для вмикання загальних перевірок (`address`, `leak`, `undefined`). Деталі і список прапорців компіляції та лінкування у файлі `meson/template-project-sanitizers/meson.build`.

Також доступна можливість вмикання перевірок виконання потоків на предмет помилок і спотворення даних (data races) наступною командою:

```
# всередині кореневої директорії проекту

meson setup build -DENABLE_SANITIZERS_THREADS=true
```

Не слід поєднувати `ENABLE_SANITIZERS` і `ENABLE_SANITIZERS_THREADS` - Meson-модуль явно поверне помилку конфігурації якщо обидві опції увімкнено, оскільки `ThreadSanitizer` несумісний з `Address+Leak` санітайзерами.

Будь-які знахідки увімкненими перевірками будуть друкуватись у стандартні потоки виводу програми (stdout/stderr).
