## Вмикання підтримки профілювання за допомогою vagrind/callgrind

Для того щоб увімкнути підтримку профілювання за допомогою `valgrind` необхідно встановити значення `true` для Meson-опції `ENABLE_CALLGRIND`:

```
# всередині кореневої директорії проекту

meson setup build --buildtype=debug -DENABLE_CALLGRIND=true
```

Після конфігурації проекту з увімкненою підтримкою профілювальника `valgrind` необхідно побудувати ціль `callgrind`:

```
# всередині кореневої директорії проекту

meson compile -C build callgrind
```

Після запуску і відпрацювання головного цільового бінарного файлу проекту, модуль `callgrind` програми Valgrind згенерує файли профілювання які можуть називатись `callgrind.out.*` і розміщуватись у директорії побудови проекту. Файли профілювання `build/src/callgrind.out.*` можуть переглядатись за допомогою програми візуального огляду файлів профілювання `kcachegrind`:

```
# всередині кореневої директорії проекту

kcachegrind build/src/callgrind.out.*
```

Визначення цілі `callgrind` розташоване у файлі `src/meson.build` (гілка `ENABLE_CALLGRIND`) - вона залежить від головного виконуваного файлу проекту і запускає `valgrind --tool=callgrind` на ньому.
