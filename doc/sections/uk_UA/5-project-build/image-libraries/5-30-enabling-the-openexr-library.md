## Вмикання інтеграції OpenEXR (EXR / HDR)

Для того щоб увімкнути бібліотеку [OpenEXR](https://openexr.com/) (формат зображень EXR з високим динамічним діапазоном) для проекту, використовуючи встановлену версію або через завантаження з мережі, необхідно встановити значення `ON` для CMake змінної `ENABLE_OPENEXR`:

```
# в середині кореневої директорії проекту

cmake -S . -B build -DENABLE_OPENEXR=ON
```

Побудова через FetchContent додатково автоматично завантажує залежність Imath. Дивись модуль `cmake/enablers/images/template-project-openexr-enabler.cmake` щодо цілі (`OpenEXR::OpenEXR`), яку слід прилінкувати до твоїх цільових об'єктів.
